# 📝 Aventura de Papel v1.0.1

¡Bienvenido a **Aventura de Papel**! Este es un proyecto de videojuego con una estética basada en bocetos en papel y logrados en Pixel Art.

## 🎨 Concepto Visual y UX
El proyecto fue diseñado buscando una experiencia de usuario inmersiva y artesanal:
* **Estética:** Dibujo lineal sobre papel.
* **Tipografía:** [Pixellari](https://www.dafont.com/pixellari.font) (estilo Pixel Art).
* **Paleta de colores:** Monocromática (blanco, negro y grises) para resaltar la textura del papel.

## 🛠️ Tecnologías utilizadas
* **Motor:** Construct 2.
* **Diseño:** Pixel Art.
* **Desarrollo:** Integración con GitHub Pages para el despliegue online.
